# AI Shopify Store Manager - Memory

## Version

v1.0

---

# Purpose

The AI Shopify Store Manager stores knowledge from previous product listings to improve future listings and maintain consistency across the Shopify store.

---

# Store Standards

Remember:

- Preferred writing style
- SEO best practices
- Brand voice
- Product page structure
- Collection organization
- Tagging conventions

---

# Successful Listings

Track products that:

- Have high conversion rates
- Receive positive customer feedback
- Rank well in search engines
- Generate repeat purchases

---

# Listing Improvements

Record:

- Customer questions
- Frequently requested information
- Better title ideas
- Better descriptions
- SEO improvements

---

# Product Categories

Remember successful structures for:

- Home & Kitchen
- Mobile Accessories
- Beauty & Personal Care
- Health & Fitness
- Baby Products
- Pet Supplies

---

# SEO Knowledge

Store:

- High-performing keywords
- Meta title patterns
- Meta description patterns
- URL structure
- Image alt text standards

---

# Learning Rules

After each published product:

- Review its performance.
- Record lessons learned.
- Improve future listings.
- Maintain consistency across the store.

# Future Integrations

Future versions may connect with:

- Shopify Admin API
- Google Search Console
- Google Analytics

- AI Knowledge Base
