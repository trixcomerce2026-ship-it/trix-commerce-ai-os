# Trix Commerce Color Palette

## Version

v1.0

---

# Purpose

Define the official brand colors for Trix Commerce.

---

# Primary Color

## Deep Blue

HEX: #1E3A8A

Meaning:

- Trust
- Reliability
- Professionalism

Usage:

- Buttons
- Headers
- Main branding elements

---

# Secondary Color

## Bright Orange

HEX: #F97316

Meaning:

- Energy
- Action
- Attention

Usage:

- Call-to-action buttons
- Discounts
- Promotions

---

# Neutral Colors

## White

HEX: #FFFFFF

Usage:

- Backgrounds
- Clean layouts

---

## Light Gray

HEX: #F3F4F6

Usage:

- Sections
- Borders
- Product cards

---

## Dark Gray

HEX: #374151

Usage:

- Text
- Footer
- Icons

---

# Color Rules

- Use Deep Blue for trust-building elements.
- Use Orange for important actions.
- Keep layouts clean and minimal.
- Avoid too many bright colors.
- Maintain consistent branding across all platforms.

---

# Mobile Design Rule

Colors should remain clear and readable on mobile devices.

---

# Brand Goal

Create a clean, modern, and trustworthy e-commerce appearance suitable for the Pakistani market.
